TPDPは、正式なproceedingsを持たないワークショップであり、既発表・投稿中・後続投稿予定の研究が混在する。そのため、TPDP2026で確認した発表については、TPDP単独の発表なのか、ICML / COLT / NeurIPS / PACMMOD / AISTATS等の既存採択・掲載済み研究の再発表・関連発表なのかを区別する必要がある。

以下は、#1〜#29について、発表元・掲載先、および主な確認元を整理したもの。

## 留意事項

- 「TPDP 2026」とあるものは、発表がTPDPに存在することを示すが、必ずしも新規・初出の研究であることを意味しない。
- TPDPはワークショップであり、既発表、投稿中、後続投稿予定の研究が混在し得る。
- 資料では、可能な限り「TPDPで見た発表」ではなく、「元会議・掲載先」を主語にする方が誤解が少ない。
- TPDPのみ確認できるものは、「TPDP2026発表情報」「preprint未確認」などと注記するのが安全。
